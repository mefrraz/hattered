🏴‍☠️ General Terms 🏴‍☠️ Termos Gerais

TermTermo DefinitionDefinição
2FA Autenticação de Dois Fatores (2FA) Two-factor authentication (2FA) is a security method requiring users to provide two different forms of identity verification to increase account security. Common methods include TOTP apps, hardware keys, and email codes. A autenticação de dois fatores (2FA) é um método de segurança que exige que o usuário forneça duas formas diferentes de verificação de identidade para aumentar a segurança da conta. Métodos comuns incluem apps TOTP, chaves físicas e códigos por e-mail.
Adblock(er) Bloqueador de Anúncios A browser extension or software that blocks advertisements on websites, enhancing browsing speed and privacy. Extensão ou software para navegador que bloqueia anúncios em sites, melhorando a velocidade de navegação e a privacidade.
Antivirus Antivírus A security program designed to detect, prevent, and remove malicious software (malware) from computers and devices. Programa de segurança destinado a detectar, prevenir e remover softwares maliciosos (malware) de computadores e dispositivos.
Archive Arquivo Compactado A single file containing multiple files or folders, often compressed to save space and facilitate transfer. Common formats include ZIP, RAR, and 7z. Um único arquivo contendo múltiplos arquivos ou pastas, geralmente compactado para economizar espaço e facilitar a transferência. Formatos comuns incluem ZIP, RAR e 7z.
Carding Carding Illegal use of stolen credit card data to make unauthorized purchases or transactions, often linked to funding piracy activities. Uso ilegal de dados de cartão de crédito roubados para fazer compras ou transações não autorizadas, frequentemente ligado ao financiamento de atividades piratas.
Crack Crack A modified version or patch of software that removes or bypasses copy protection or licensing restrictions. Versão modificada ou patch de software que remove ou contorna proteções contra cópia ou restrições de licença.
DDL (Direct Download Link) Link de Download Direto (DDL) A direct URL link that allows downloading files from a server over HTTPS without needing peer-to-peer sharing. Um link URL direto que permite baixar arquivos de um servidor via HTTPS sem necessidade de compartilhamento entre usuários.
DMCA takedown notice Notificação de Remoção DMCA A formal legal request under the Digital Millennium Copyright Act to remove infringing content from websites or online services. Pedido legal formal, previsto na Lei de Direitos Autorais dos Estados Unidos (DMCA), para remoção de conteúdo que infringe direitos autorais de sites ou serviços online.
DNS Sistema de Nomes de Domínio A protocol that translates domain names (like www.example.com) into IP addresses, facilitating internet navigation. Custom DNS services can be used to bypass blocks or increase privacy. Protocolo que traduz nomes de domínio (como www.exemplo.com) em endereços IP, facilitando a navegação na internet. Serviços DNS personalizados podem ser usados para contornar bloqueios ou aumentar a privacidade.
Download Manager Gerenciador de Downloads Software that improves downloading by increasing speed, enabling pause/resume, and managing multiple downloads, commonly used with DDLs. Programa que melhora o download ao aumentar a velocidade, permitir pausar e retomar, e gerenciar múltiplos downloads, frequentemente usado com links diretos (DDL).
False Positive Falso Positivo When antivirus software incorrectly flags a safe file as malicious, often happening with cracked software or keygens. Quando o antivírus classifica incorretamente um arquivo seguro como malicioso, situação comum com softwares cracks ou keygens.
Firewall Firewall A network security system that monitors and controls incoming and outgoing traffic based on predefined security rules to protect internal networks from threats. Sistema de segurança de rede que monitora e controla o tráfego de entrada e saída com base em regras de segurança para proteger redes internas contra ameaças.
FTP Protocolo de Transferência de Arquivos (FTP) A protocol used for transferring files between computers over a network. Frequently used in warez servers for sharing pirated content. Protocolo usado para transferir arquivos entre computadores numa rede. Frequentemente utilizado em servidores warez para compartilhamento de conteúdo pirata.
IRC Internet Relay Chat (IRC) A text-based chat protocol designed for real-time group communication and discussion, often used in warez communities. Protocolo de chat baseado em texto, projetado para comunicação em grupo em tempo real, frequentemente usado em comunidades warez.
Keygen Gerador de Chaves (Keygen) A program that generates valid product keys or serial numbers to activate software illegally. Programa que gera chaves de produto ou números de série válidos para ativar software de forma ilegal.
NFO Arquivo NFO A text file accompanying pirated releases containing information about the release, such as software title, group name, and release details. Arquivo de texto que acompanha lançamentos piratas contendo informações sobre o lançamento, como título, grupo responsável e detalhes da release.
Patch Patch A modification applied to software to remove protections or fix bugs, often used to bypass licensing in pirated software. Modificação aplicada a um software para remover proteções ou corrigir falhas, frequentemente usada para contornar licenças em software pirata.
Peer-to-Peer (P2P) Rede Ponto a Ponto (P2P) A decentralized network where users share files directly with each other without a central server, widely used for distributing pirated content. Rede descentralizada onde usuários compartilham arquivos diretamente entre si, sem servidor central, muito usada para distribuir conteúdo pirata.
Proxy Proxy An intermediary server that routes user requests and responses, often used to mask IP addresses and bypass restrictions. Servidor intermediário que encaminha requisições e respostas dos usuários, usado para mascarar endereços IP e contornar restrições.
Ripping Ripagem The process of extracting digital media (audio, video) from physical media like CDs or DVDs into files for easier distribution and playback. Processo de extração de mídia digital (áudio, vídeo) de mídias físicas como CDs ou DVDs para arquivos digitais, facilitando distribuição e reprodução.
Scene Release Release da Scene A pirated digital release officially distributed by a group within The Scene, often considered the “original” copy in warez circles. Lançamento pirata distribuído oficialmente por um grupo da Scene, geralmente considerado a cópia “original” nos círculos warez.
Seed / Seeder Seed / Seeder A user in a P2P network who shares the complete file with others, allowing downloads to continue. Usuário em uma rede P2P que compartilha o arquivo completo com outros, permitindo que os downloads continuem.
The Scene The Scene An underground network of groups specializing in cracking and releasing pirated digital media worldwide. Rede clandestina de grupos especializados em crackear e lançar mídia digital pirata mundialmente.
Torrent Torrent A file containing metadata used by BitTorrent clients to download and share files over P2P networks. Arquivo contendo metadados usados por clientes BitTorrent para baixar e compartilhar arquivos em redes P2P.
Usenet Usenet A decentralized system of servers allowing users to post and download messages and files, often accessed via subscriptions and NZB files for piracy content. Sistema descentralizado de servidores que permite postar e baixar mensagens e arquivos, frequentemente acessado via assinaturas e arquivos NZB para conteúdo pirata.
VPN Rede Privada Virtual (VPN) A service that creates an encrypted tunnel between your device and a remote server, hiding your IP and enhancing privacy, often used to bypass geo-restrictions and protect identity during piracy. Serviço que cria um túnel criptografado entre seu dispositivo e um servidor remoto, ocultando seu IP e aumentando a privacidade, frequentemente usado para contornar restrições geográficas e proteger a identidade durante a pirataria.
Warez Warez Pirated software, games, music, or media distributed illegally. Software, jogos, músicas ou mídia pirateados distribuídos ilegalmente.
